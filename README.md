# homebrew-stoic-public
Homebrew tap for stoic public formulae

## How do I use it ?
`brew install sutoiku/stoic-public/<formula>`

Or `brew tap sutoiku/stoic-public` and then `sutoiku/stoic-public/<formula>`.
